# ASSETS Directory

## Purpose

This directory contains supporting assets for the Q100 Concept Baselines documentation, including:

- **Geometry files:** CAD models, STEP/IGES files
- **Aerodynamic data:** CFD results, wind tunnel data, drag polars
- **Performance data:** Mission profiles, range-payload curves
- **Mass breakdowns:** Detailed weight spreadsheets
- **Diagrams and figures:** Concept sketches, technical illustrations
- **Images:** Renderings, photographs, screenshots

## Organization

Assets are organized by document reference:

```
ASSETS/
├── 01_Concept_Overview/
├── 04_Geometry_Definition/
├── 05_Planform_Layout/
├── 09_Mass_Breakdown/
└── ...
```

## File Naming Convention

- Use descriptive names: `Q100_planform_layout_v1.0.png`
- Include version numbers for iterative files
- Reference source document: `LC-11_0010-05_planform_top_view.svg`

## Supported Formats

- **CAD:** STEP, IGES, STL
- **Images:** PNG, SVG, JPG
- **Data:** CSV, XLSX, JSON
- **Documents:** PDF (for external references)

## Access Control

- **Internal Use Only:** All assets are proprietary AMPEL360 intellectual property
- **Distribution:** Controlled by Architecture Board and Document Control
